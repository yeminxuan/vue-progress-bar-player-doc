## 安装

### 依赖环境

- 包管理器，如[pnpm](https://pnpm.io/zh/)、[yarn](https://classic.yarnpkg.com/en/)、[npm](https://www.npmjs.com/)等。

::: tabs

@tab pnpm

```bash
pnpm add vue3-progress-bar
```

@tab yarn

```bash
yarn add vue3-progress-bar
```

@tab npm

```bash
npm i vue3-progress-bar
```

:::
